# 1️⃣ 개념 정리 

## 01. 데이터 분석이란

데이터 분석: 유용한 정보를 발견하고 결론을 유추하거나, 의사 결정을 돕기 위해 데이터를 조사, 정제, 변환, 모델링하는 과정 / 올바른 의사결정을 돕기 위한 통찰 제공 / 비교적 소규모 
데이터 과학: 통계학, 데이터 분석, 머신러닝, 데이터 마이닝 등을 아우르는 개념 / 문제 해결을 위한 최선의 솔루션 / 대규모 

데이터 분석- 기술통계 / 탐색적 데이터 분석 / 가설검정
- 기술통계: 관측이나 실험을 통해 수집한 데이터를 정량화하거나, 요약하는 기법. 평균 계싼 최솟값 최댓값 찾기
- 탐색적 데이터 분석: 데이터를 시각적으로 표현하여 주요 특징을 찾고 분석하는 방법
- 가설검정: 주어진 데이터를 기반으로 특정 가정이 합당한지 평가하는 통계 방법.

분석도구
프로그래밍 언어: 파이썬, R
데이터가 데이터베이스 형태로 있다면 SQL 사용 가능

파이썬 필수 패키지
- 넘파이: 배열을 위한 수학 패키지
- 판다스: 엑셀의 시트처럼 숫자와 문자를 섞어 표 형태로 저장할 수 있는 데이터프레임을 사용
- 맷플롯립: 파이썬 데이터 시각화. 정적 동적 그래프
- 사이파이: 넘파이 기반으로 구축된 계산 전문 수학 과학 패키지
- 사이킷런: 파이썬의 독보저인 머신러닝 패키지. 넘파이와 사이파이에 크게 의존.


데이터 마이닝: 데이터에서 패턴 혹 지식을 추출하는 작업. ex. 머신러닝, 통계학, 데이터 베이스 시스템
머신러닝: 데이터에서 자동으로 규칙을 학습하여 문제를 해결하는 소프트웨어를 만드는 기술. ex. 딥러닝
머신러닝은 규칙이나 패턴을 사용하는 주체가 사람이 아니라 컴퓨터
데이터 마이닝 알고리즘은 규칙을 만드는 과정이 투명 / 머신러니 알고리즘은 알기 어려움


## 02. 구글 코랩과 주피터 노트북

구글 코랩-웹 브라우저에서 무료로 파이썬 프로그램 테스트하고 저장할 수 있는 서비스  
텍스트셀: 코랩에서 실행할 수 있는 최소 단위. HTML, 마크다운 혼용 사용 가능
코드 셀: 파이썬 코드를 입력하고 실행할 수 있는 셀 / 일반 프로그램과 달리 셀 단위로 코드 실행 가능하여 이전 셀을 다시 실행할 수 있음.   
노트북: 코랩 노트북으로 동시에 사용할 수 있는 구글 클라우드의 가상 서버는 최대 다섯 개. 한 개의 노트북은 12시간까지 실행 가능. 

깃허브: 여러 사람이 동시에 코드 수정 가능. 

마크다운
굵게 쓰기- **마크다운**    
기울임꼴- *마크다운* / _마크다운_  
취소선- ~~마크다운~~   
코드서체- 'print("Hello world!")'  
들여쓰기- >마크다운  
글머리기호- *마크다운 / -마크다운  
하이퍼링크- [마크다운](인터넷주소)  



## 03. 이 도서가 얼마나 인기가 좋을까요?
csv 파일  
콤마로 구분된 텍스트 파일   
한 줄이 하나의 레코드. 레코드는 콤마로 구분된 여러 필드로 구성   
행은 한 줄로 표현. 열은 콤마로 구분  

파일 인코딩 형식 확인하기: chardet.detect() 함수  
인코딩: 문자를 컴퓨터가 이해할 수 있는 0과 1의 이진형태로 바꾸는 것  
- UTF-8은 모든 문자를 컴퓨터에 표현하기 위해 만들어진 유니코드를 인코딩하는 방식 중 하나로 최대 4바이트까지 가능.  
-EUC-KR은 한글을 위한 완성형 인코딩 중 하나로 2바이트 사용   

판다스는 csv 파일을 읽어 데이터프레임이라는 표 형식 데이터로 저장  
판다스에는 데이터프라임과 시리즈라는 데이터 구조 존재  
판다스 시리즈: 1차원이 배열과 비슷. 시리즈에 담긴 데이터는 모두 동일한 종류 ex. 정수, 문자열  
판다스 데이터프레임: 2차원 배열과 비슷. 열마다 다른 데이터 타입(ex. 문자열/정수) 가능. 대신 데이터프레임의 같은 열에서는 같은 데이터 타입이어야 함.   

csv 파일을 데이터프레임으로 읽기: read_csv() 함수  
먼저 imoport pandas as pd (여기서 pandas를 pd로 줄여 씀)  
df=pd.read_csv()  

low_memory=False <- 파일을 나누어 읽지 않고 한 번에 읽는 것.   
이 경우에 파일을 한 번에 모두 읽기에 많은 메모리를 사용하여 메모리 부족 오류 발생 가능성 있음  
그래서 dtype 매개변수를 사용하는 방법도 있음 ex. dtype={'ISBN':str, '세트 ISBN':str, '주제분류번호': str}  

첫 번째 열: 데이터프레임의 인덱스. 0부터 시작하는 인덱스 번호 자동 부여  
첫 번쨰 행: 열 이름.   

데이터프레임을 csv 파일로 저장하기: to_csv() 메서드  
이 코드는 기본적으로 UTF-8 형식으로 저장.

# 3️⃣ 확인 문제

## 문제 1.

> **🧚Q. 데이터 분석의 개념을 넓은 의미와 좁은 의미로 나누어 설명하고, 두 개념의 차이점도 함께 정리해주세요.**

데이터 분석의 좁은 의미는 유용한 정보를 발견하고 결론을 유추하거나 의사 결정을 돕기 위해 데이터를 조사, 정제, 변환, 모델링 하는 과정으로 정의한다. 비즈니스 결정을 과학적으로 내리기 위한 도구로 사용되는 경향이 있다. 
대이터 분석의 넓은 개념은 데이터 과학으로 문제 해결을 위하여 최선의 솔루션을 만드는 일을 말한다. 
데이터 분석과 데이터 과학의 차이는 최종적으로 만들어내는 결과물에 있다. 데이터 분석의 경우 올바른 의사 결정을 위한 통찰을, 데이터 과학은 최선의 솔루션을, 결국 데이터 과학이 데이터 분석보다 더 멀리 보는 결과물을 산출한다는 것을 알 수 있다.
